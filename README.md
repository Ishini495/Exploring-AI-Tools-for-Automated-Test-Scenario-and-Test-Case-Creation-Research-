# Exploring-AI-Tools-for-Automated-Test-Scenario-and-Test-Case-Creation-Research-
This research explores AI tools that help QA engineers automatically generate test scenarios and test cases. 
It focuses on **free**, **easy-to-use tools with fast responses**, including **ChatGPT**, **Claude**, **Perplexity**, **DeepSeek**, **Omniit**, **CloudQA**, and **Kualitee**, to improve test coverage and reduce manual testing effort.
